# Avance (Prototipo)

Se leyeron los documentos:

- `Levantamiento inicial.docx`
- `documetos prototipo.docx`

Y se genero un prototipo web navegable en `prototipo/` con las pantallas definidas:

- Login
- Dashboard
- Inventario
- Escaneo de productos (input tipo pistola lectora)
- Historial
- Configuracion

## Como verlo

En `prototipo/`:

- `cmd /c "npm install"`
- `cmd /c "npm run dev"`

Luego abrir `http://127.0.0.1:5173/`.
